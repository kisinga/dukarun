On-device recognition spike

Measures whether nearest-match separates your real products. Pass = false-accept ≤2%, genuine accept ≥85%, embed ≤~100ms. No app code — throwaway.

Pick one dataset folder. Each subfolder is one item (named with the product id) and holds that product's photos:

dataset/<productId>/photo1.jpg, photo2.jpg …

Use varied photos per item (lighting, angle, distance, background). The harness holds some out to test against. Include 2–3 look-alike items.
